If Arrays.sort is used to sort non-English text, then an instance of Collator must be passed to insure correct linguistic sorting. Be sure to pass the correct user-preferred locale to the Collator instantiation.

Noncompliant Code Example

  Arrays.sort(someText);

Compliant Solution

  Locale userPreferredLocale = Locale.forLanguageTag("zh-Hans");
  ...
  Arrays.sort(someText, Collator.getInstance(userPreferredLocale));
