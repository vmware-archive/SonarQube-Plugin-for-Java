Hard-coded (English or non-English) Locale field use should be avoided. Use of these constants could prevent software from running correctly. Always use an explicit, user-preferred locale as an argument for any Locale parameter.

Noncompliant Code Example

  text.toLowerCase(Locale.CHINA);

Compliant Solution

  Locale userPreferredLocale = Locale.forLanguageTag("zh-Hans");
  ...
  text.toLowerCase(userPreferredLocale);
