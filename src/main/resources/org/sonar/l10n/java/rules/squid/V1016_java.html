If Collections.sort is used to sort non-English text, then an instance of Collator must be passed to insure correct linguistic sorting. Be sure to pass the correct user-preferred locale to the Collator constructor.

Noncompliant Code Example

  Collections.sort(someText);

Compliant Solution

  Locale userPreferredLocale = Locale.forLanguageTag("zh-Hans");
  ...
  Collections.sort(someText, Collator.getInstance(userPreferredLocale));
